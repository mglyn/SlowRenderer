## 技术点
- 向量和矩阵计算库；
- 线程池并行；
- 多边形裁剪与直线绘制；
- 背面剔除；
- 半平面交渲染三角形；
- 深度缓冲、修正属性插值；
- Blinn-Phong光照模型；
